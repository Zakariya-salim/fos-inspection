FOS Vehicle Inspection System - Offline Test Build

This version is ready to open without PHP, MySQL, Apache, XAMPP, or any server.

How to run:
1. Extract the ZIP file.
2. Open index.html in Chrome, Edge, or Safari.
3. Use the Home button to return to index.html.
4. Use Admin Login to access the dashboard.

Default test accounts:
Super Admin: admin@fos.local / Admin@123
Unit Admin: unit@fos.local / Unit@123
Viewer: viewer@fos.local / Viewer@123

Important:
- This offline build stores data in the browser localStorage.
- Data is only saved on the same device/browser.
- For shared database testing across devices, use the PHP + MySQL production version.
