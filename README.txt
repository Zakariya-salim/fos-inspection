FOS Vehicle Inspection System

Open index.html to start.

Pages:
- index.html        Home
- login.html        Admin / unit login
- public-form.html  Public no-login inspection form
- dashboard.html    Admin dashboard
- report.html       Report viewer

Demo credentials:
- Super Admin: admin@fos.local / Admin@123
- Unit Admin: unit@fos.local / Unit@123
- Unit Viewer: viewer@fos.local / Viewer@123

Notes:
- Data is stored in browser local storage as a mini demo database.
- Super Admin can add/delete units and accounts.
- Unit Admin can manage accounts/questions for their unit.
- Public users do not log in.
